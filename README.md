### 1. 게임설명
마우스 버튼을 클릭하여 장애물을 넘는 달리기 게임입니다.   
허접한 게임이지만 언젠가 강화해서 모바일 게임으로 낼 계획입니다.  
Pocessing 프로그램이 없으신 경우 위의 패키지파일을 다운받아서 실행하면 됩니다.

### 2. 개발환경
Processing 3.4, Sound library, 사운드는 무료음원 사용.
